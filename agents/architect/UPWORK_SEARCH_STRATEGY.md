## ðŸš© RED FLAG Signals (AI-Overstaters)

| Red Flag | Detection Method |
|----------|------------------|
| **"AI Expert" but no GitHub** | Claims AI/ML but zero code samples |
| **Vague tech stack** | Says "full stack" but can't name frameworks |
| **Copy-paste proposals** | Uses generic templates ("I am a HubSpot expert...") |
| **No verifiable portfolio** | "I can't share due to NDA" for EVERY project |
| **Overpromising timeline** | "I can do this in 2 days" for complex work |
| **No technical interviews** | Refuses live coding or architecture discussion |
| **Rate too low for scope** | <$50/hr for enterprise HubSpot = inexperienced or ghosting risk |
| **Profile photo = AI-generated** | Not a reliable indicator alone, but correlates with fake profiles |
| **Zero LinkedIn or LinkedIn doesn't match Upwork** | Inconsistent identity |
| **"I know everything about HubSpot"** | HubSpot is huge - no one does. Specificity matters. |
